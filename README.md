pjs2png
=======
Processing.js script -> png

Prerequisites
-------------
### Ruby & Sinatra
http://www.ruby-lang.org/

    gem install sinatra

### PhantomJS
http://www.phantomjs.org/


Usage example
-------------

    git clone git://github.com/hiroshi/pjs2png.git
    cd pjs2png
    ./pjs2png /path/to/phantomjs examples/gear.pjs gear.png
